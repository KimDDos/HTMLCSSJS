for, while, do~while : java와 동일한 문법을 사용

  • for : 반복횟수가 명확할 때 사용
    for(초기값; 조건식; 증감식;){
    실행문;
    }
  • while, do~while : 반복 횟수가 명확하지 않을 때
    while(조건식){
    실행문;
    }

    do{
    실행문;
    }
    while(조건식);
  • 모든 조건식은 참(true)일 때 실행
  • 조건이 계속 true라면 무한 루프에 걸릴 수 있음
    조건이 실행문안에서 false로 바뀔수 있는 구문이 반드시 있어야 함.
  • break; 구문을 통해 빠져 나갈수 있음
