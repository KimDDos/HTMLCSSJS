함수 : 특정 기능을 수행하기 위해 작성한 코드 - 호출되어야만 실행 자바에서는 클래스안에 메서드가 생성, 자바스크립트는 어디서든 function 생성 가능

  • function 이라는 키워드를 사용하여 정의
  • function gkatnaud( 파라미터 ){
    로직작성;
    }
  • 함수명 : 기능을 함축하는 의미로 작성
  • 파라미터(매개변수) : 선택적으로 사용 (없어도 ()는 있어야함)
  • 자바스크립트에서 파라미터는 타입이 자유도가 높아서 다양한 형태로 사용 가능
  • return 함수 종료 역할과 동시에 특정 결과값을 호출한 위치로 반환
