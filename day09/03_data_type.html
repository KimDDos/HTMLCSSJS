변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number => 정수, 실수, NaN(not a Number : 숫자가 아님)
  • 문자 : String = "", '', `` 감싼 값들
  • 불리언 : Bollean => true / false
  • 객체 : Object => 1개 이상의 데이터가 특정한 구조를 가지고 있는 형태 {키 : 값} ({ } : 아무것도 없는 사실도 1개의 객체로 취급)
  • 배열 : Array => [1,2,3,...] 형태의 데이터 집합 (순서, 길이가 있음)
  • 알 수 없는 값 : null => 뭔가 있는것 같긴한데 파악이 안됨
  • 미지정된 값 : undefined => 값이 할당되지 않음
  • 데이터의 타입을 확인하는 명령어 : typeof 변수

변수의 관리

  1. 기본자료형 : 값의 메모리 주소값이 가르키는 곳에 직접 할당되는 값들 number, boolean, String(참조자료형이지만 사용편의상 기본자료형과 같이 사용)
  2. 참조자료형 : 값이 저장되어 있는 주소를 할당 string, object, array, funtion, null, undifined
  3. == : 같다.(값만 비교) | != : 같지 않다. === : 같다(값과 타입이 같은지 비교) !== : 같지않다.(값과 타입이 같지 않은지 비교)
