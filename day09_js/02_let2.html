자바 스크립트의 변수 사용

  • 값을 담고 있는 메모리 공간에 이름을 붙여서 활용하기 위함
  • 자바스크립트에서 변수는 선언하는 역할만 함 (이름 정의에 집중)
  • var, let(ES6 부터는 let을 사용하는 것을 적극 권장), const
  • 자바와 달리 변수에 자료형을 표시하지 않음.
  • let으로 선언하면 어떤 값이든 다 담을 수 있음
  • var name = 'hong'; var name = 'kim'; => posible
  • 자바에서의 지역변수 개념
  • let 중복 선언 안 됨, 재할당은 가능 let name = 'hong'; name = 'kim'; // 재할당은 가능
  • 스코프(scope : 사용범위) { } 에서만 사용가능 스코프를 벗어나면 재선언 가능
  • 지역변수 스코프 : { }
  • const : 재선언/재할당 불가능 const name : 'hong'; name = 'kim'; // 불가능.
  • 선언 이후 항상 고정값을 가지는 경우
  • 고정된 값을 사용하는 규칙, 참조, 룰, 객체, 엘리먼트 정의시 사용
