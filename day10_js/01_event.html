이벤트 객체는 전역객체(전체영역에 반영시키는 객체)로써 모든 태그에 객체를 참조시킬 수 있다.

  • a tag와 button tag에 이벤트를 주로 참조시켜 클릭에 대한 반응을 기대할수 있게 만드는게 일반적
  • a htef 속성에 의해 자체 이동(동기화 방식)이 구현되어서 별도의 이동을 막는 코드가 필요
    #을 달아둔다던지 하는 조치가 필요함.
  • 객체에 이벤트를 부여하는 방식에서는 크게 3가지가 있다. (각자 목적과 작동방식이 다르다.) 1. document.addEventListener
  • 화면의 출력영역을 document영역 (객체) 이라고 한다.
  • HTML 태그는 html element, 속성은 attribute, 일반 텍스트 textNode 라고 함
  • html로 출력하기 : innerHTML이라는 property를 사용
  • text로 출력하기 : innerText property를 사용
  • html element를 id값으로 객체 가져오기 : getElementById('idName')
  • html element를 css 선택자를 사용하여 가져오기 : querySelector('selector') . / # / input type="text" 등 ...
