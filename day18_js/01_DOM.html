DOM(Document Object Model) : html(문서) 객체 모델

  • html 하위 태그는 각각의 기능(method), 속성(properties)를 갖고 있는 문서 객체이다.
  • element 가져오기 : tag element를 가져오는 메서드 byId, byClass, byTagName
  • document.getElementById('idName') : 단일 엘리먼트 리턴
  • document.getElementsByClassName('className') : 복수 엘리먼트 리턴
    => HTMLCollection 타입으로 리턴
  • document.getElementsByTagName('tagName') : 복수 엘리먼트 리턴
    => HTMLCollection 타입으로 리턴
  • 복수 엘리먼트로 가져오게 되면 index를 선택하여 사용
