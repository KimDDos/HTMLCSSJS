  • input, textarea, select, html element 속성, 텍스트 데이터 받아오기

  • value 속성의 값은 모두 string 처리 됨,
  • string 처리된 value값을 숫자로 사용하기 위해서는 변환이 필요 parseInt를 사용하면 됨, 자바랑 똑같음
